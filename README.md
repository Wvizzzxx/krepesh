## Быстрый старт

### Что нужно установить

1. **Node.js** (версия 18 или выше)
   - Скачать: https://nodejs.org/
   - При установке поставьте галочку "Automatically install the necessary tools"

2. **pnpm** (менеджер пакетов)
   - После установки Node.js выполните в командной строке:
   ```
   npm install -g pnpm
   ```

3. **MongoDB** (база данных)
   - Скачать: https://www.mongodb.com/try/download/community
   - Установить с настройками по умолчанию
   - **Важно:** при установке поставьте галочку "Install MongoDB as a Service"

### Пошаговая установка

```bash
# 1. Скопируйте проект на компьютер (или распакуйте архив)

# 2. Откройте командную строку в папке проекта
#    (в проводнике: правой кнопкой → "Открыть в PowerShell" или "Открыть в терминале")

# 3. Установите зависимости
pnpm install

# 4. Заполните базу данных тестовыми данными
pnpm seed

# 5. Запустите все сервисы
#   双击 файл start-all.bat
#    Или введите в терминале:
pnpm dev
```

### Готово! Откройте в браузере:

| Сервис | Адрес |
|--------|-------|
| 🌐 Сайт магазина | http://localhost:3000 |
| ⚙️ Админ-панель | http://localhost:3002 |
| 🔌 API сервер | http://localhost:3001 |

## Запуск проекта

### Способ 1: Все сразу (рекомендуется)

Дважды кликните на файл **`start-all.bat`** в корне проекта.

### Способ 2: Через терминал

```bash
# Запуск всех сервисов одновременно
pnpm dev

# Или запуск отдельных сервисов:
pnpm dev:api      # Только API сервер
pnpm dev:web      # Только сайт
pnpm dev:admin    # Только админка
```

### Способ 3: Через Visual Studio Code

1. Откройте папку проекта в VS Code
2. Откройте встроенный терминал (Ctrl + `)
3. Выполните:
```bash
pnpm dev
```

### Если MongoDB не запускается

Убедитесь, что служба MongoDB запущена:

**Windows:**
```bash
# Проверка статуса службы
sc query MongoDB

# Запуск службы (от администратора)
net start MongoDB
```

**Или через PowerShell (от администратора):**
```powershell
Start-Service MongoDB
```

## Настройка email

Для отправки писем подтверждения email используется Яндекс почта через SMTP.

### Настройка:

1. Зайдите в настройки Яндекс почты → **Пароли приложений**
   - Прямая ссылка: https://passport.yandex.ru/profile/security/application-passwords
2. Создайте новый пароль приложения (название: "КРЕПЁЖ")
3. Скопируйте полученный пароль
4. Откройте админ-панель → **Настройки Email**
5. Введите:
   - **Email Яндекса:** ваш-email@yandex.ru
   - **Пароль приложения:** скопированный пароль
6. Нажмите **Сохранить** → **Тестовое письмо**

## Решение проблем

### "MongoDB не найден"
Установите MongoDB Community Server: https://www.mongodb.com/try/download/community

### "pnpm: команда не найдена"
```bash
npm install -g pnpm
```

### "Порт уже занят"
```bash
# Найдите процесс на порту и завершите его:
netstat -ano | findstr :3001
taskkill /F /PID <номер_процесса>
```

### "Ошибка подключения к MongoDB"
Убедитесь, что служба MongoDB запущена:
```bash
net start MongoDB
```

### База данных пуста / товары не отображаются
```bash
pnpm seed
